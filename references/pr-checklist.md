# PR Review Checklist

## 功能正确性
- [ ] 代码实现了需求描述的所有功能
- [ ] 边界条件处理完整（空值、0、负数、极限值）
- [ ] 错误处理不吞异常

## 代码质量
- [ ] 命名清晰，无单字母变量
- [ ] 函数不超过 80 行
- [ ] 嵌套不超过 3 层
- [ ] 无重复逻辑
- [ ] 无硬编码魔法数字

## 安全性
- [ ] 无 SQL 注入（参数化查询）
- [ ] 无 XSS（用户输入已转义）
- [ ] 无硬编码密钥/密码
- [ ] 权限校验完整

## 性能
- [ ] 无 N+1 查询
- [ ] 无循环内数据库/网络调用
- [ ] 无不必要的深拷贝

## 测试
- [ ] 正常路径有测试
- [ ] 异常路径有测试
- [ ] 边界值有测试

## 文档
- [ ] 公共 API 有文档注释
- [ ] README 已更新（如有接口变更）
- [ ] 复杂逻辑有行内注释
